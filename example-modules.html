Example Modules

Reusable components to jumpt start development.

This whole page here is in a .page module. This is a module created for basic pages with a wrapping .page div, and a .page__header and optional .page__footer.

Note that these components use basic inline javascript for some UI actions. Since Skyline focuses on CSS, you are free to use any javascript libraries you wish.

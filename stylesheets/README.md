COLOPHON & GUIDELINES
---------------------------------------------------------

Skyline is an object-oriented CSS architecture system
written in SCSS with BEM notation and is heavily inspired
by the concepts found in SMACCS, InuitCSS, Bootstrap, &
Atomic Design.
